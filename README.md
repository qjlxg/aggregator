### 🛠️ 快速上手 (Quick Start)

#### 1. Clash 系列 (Windows / Android / macOS / iOS)
1. 复制上面的 **Clash 订阅** 链接。
2. 打开 Clash 客户端，进入 `Profiles`。
3. 粘贴链接并点击 `Download`。
4. 在 `Proxies` 面板中选择 **🔰 节点选择** 或 **🚀 自动测速**。

#### 2. v2rayN / v2rayNG
1. 复制上面的 **V2Ray 订阅** 链接。
2. 在客户端选择“从剪贴板导入”或添加订阅分组。
3. 执行“更新订阅”。

### 📡 节点详情
* **更新频率**:  不定时更新 。
* **地理识别**: 基于 `MaxMind GeoLite2` 数据库。
* **支持协议**: SS, VMess, VLESS, Trojan, Hysteria2。
